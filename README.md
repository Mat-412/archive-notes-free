# Archive Notes

A powerful note-taking application with Markdown and LaTeX support, featuring real-time preview, hierarchical note organization, and built-in tools for mathematical notation.

## Features

- **Markdown Support**: Write notes using Markdown syntax with real-time preview
- **LaTeX Integration**: Full LaTeX support for mathematical expressions and equations
- **Built-in LaTeX Symbol Insertion**: Searchable library of mathematical symbols (Press F7)
- **Custom Table Creation**: Easy-to-use table builder with customizable rows and columns (Press F8)
- **Real-time Preview**: See your formatted content instantly as you type
- **Hierarchical Organization**: Organize notes in notebooks with nested sub-notes
- **Export/Import Functionality**: Save and load your notes in JSON format
- **Customizable Background**: Set custom background colors for a personalized experience (Press F6)
- **Print Support**: Print your notes directly from the application

## Getting Started

### Interface Overview

The app consists of three main sections:

- **Sidebar**: Contains your notebooks and notes hierarchy
- **Editor Pane (Left)**: Where you type Markdown and LaTeX content
- **Preview Pane (Right)**: Shows the formatted result of your content

### Creating Notes

1. Click "Add Notebook" to create a top-level notebook
2. Use the "+" button to add notes within notebooks
3. Click on a note header to select and edit it
4. Type your content in the editor pane using Markdown and LaTeX syntax
5. View the formatted result in the preview pane

### Sidebar Controls

- **"Add Notebook" button**: Creates a top-level notebook

### Organizing Notes

- **Reordering**: Drag a note and drop it when the blue guide line appears in the desired spot
- **Nesting**: Drag a note and drop it when the blue guide box appears around the desired parent note

## Markdown Basics

Archive Notes supports standard Markdown syntax:

- **Headers**: `# H1`, `## H2`, `### H3`, etc.
- **Bold**: `**bold text**`
- **Italic**: `*italic text*`
- **Lists**: `- item` or `1. item`
- **Links**: `[text](url)`
- **Code**: `` `inline code` `` or code blocks with triple backticks

## LaTeX Usage

LaTeX is used for mathematical expressions and symbols:

### Basic Syntax

- Inline math: `$expression$` (e.g., `$\theta$`)
- Display math: `$$expression$$` (e.g., `$$\int_0^1 x^2 dx$$`)

### Inserting Symbols

Press `F7` to open the LaTeX symbol insertion tool. Browse or search for symbols, then click to insert them at your cursor position.

## Creating Tables

Press `F8` to open the custom table builder:

1. Specify the number of rows (1-20) and columns (1-10)
2. Fill in the table cells (first row is treated as headers)
3. Click "Confirm" to insert the table in Markdown format
